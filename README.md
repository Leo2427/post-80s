## 80后人口数量

80后（1980年至1989年出生）的出生人口约为‌2.16亿人‌，不同统计口径下估计值在2.16亿至2.23亿之间。截至2025年，80后群体的岁数在36-45岁之间，存活人口估计在‌2.10亿至2.12亿‌之间。粗略计算，有数百万甚至上千万的80后已经离世了。

## 80后的负债

80后的人均负债水平较高，例如2025年3月的数据显示，80后人均负债达到38.7万元，约为全国平均水平的1.7倍。80后债务三座大山：房贷占比约65%、车贷占比 12%、子女教育贷占比 8%，这三项 “刚性债务” 占总负债的 85%，剩下的 15% 则是信用卡分期、消费贷（用于日常开支、老人医疗）。网络上很多80后表示太累了，下辈子不来了。然而每个时代都有每个时代的上甘岭，重要的是一路向前，让人类延续下去。

## 80后的童年

80后的童年，回不去的美好，那个时代没有电脑，没有手机，确让80后拥有最快乐的童年：
- 那个时候都是同村孩子成群结队的走路去上学，不需要父母接送的；
- 穿的都很破旧，有些是有补丁的，最记忆犹新的就是塑料凉鞋了，剪掉后跟带子可以变拖鞋；
- 从砖厂，瓦厂偷泥巴来玩，这种泥巴特别细腻，没有杂质；
- 男孩女孩都会玩跳皮筋，女孩子跳的更好一些；
- 弹玻璃珠子，已经忘了这些珠子最初是从来里来的了，反正就是莫名其妙的就有；
- 玩老鹰捉小鸡，有时候跑的太猛，母鸡队伍最后那个人总是被甩出来；
- 玩捉迷藏，藏在各种地方，树上，稻草堆里；
- 骑的单车大多都是28大杠，只有少数女士单车；
- 乒乓球台是水泥做的，中间用断砖头拦起来打；
- 还有一种普通话说不出来名字的东西，我们那叫人工拐，是一种几厘米大小的长方形小纸片，上面有动画人物，可以放地上拍，拍翻了可以赢走；
- 有一种游戏，一根绳子穿在手指尖，可以做出降落伞、五角星等形状，另外一个人需要找到接下去的方法；
- 东南西北，一种用纸做的小玩具；
- 艾草叶捆起来成一个球形，用来扔站在中间的人，接住一次多加一条命
- ......

## 80后的工作

80后当前工作现状呈现严峻挑战，主要表现为就业压力加剧、收入缩水、职业发展受阻及适应性转型需求：
- 就业压力与年龄歧视
  - 求职困难：35岁以上求职者普遍被年龄歧视
  - 裁员风险：大厂裁员中，35岁以上占大多数
- 行业转型与收入困境
  - 收入缩水：各行业降薪普遍
  - ‌职业转型‌：失业者中88%转向外卖、快递、网约车行业
    
失业后消失的80后，这群曾经被贴上“奋斗一代”标签的人，正集体陷入一种矛盾，失业后，他们中的许多人似乎“不想找工作了”。希望社会各界人士正视这一现象，给80后一个体面的退场方式......
